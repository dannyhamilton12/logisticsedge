# UK Import Duty Calculator Quick Reference Guide

## What This Calculator Does

This calculator helps UK importers determine the total cost of importing goods into the UK, including:
- Import duty (calculated based on commodity code and origin)
- Import VAT (20% standard, 5% reduced, or 0%)
- Total landed cost (CIF value + duty + VAT)

## Key Terms

**CIF Value** - Cost + Insurance + Freight. This is the customs value used to calculate duty.

**Commodity Code** - The 6-10 digit HS/tariff code that determines the duty rate. Search at trade-tariff.service.gov.uk

**Duty Rate** - The percentage of duty applied to the CIF value. Varies by product and origin.

**Preferential Rate** - Reduced or zero duty under a UK trade agreement (TCA with EU, or FTAs with other countries).

**Import VAT** - VAT charged on imports. Usually 20%, but some goods qualify for 5% or 0%.

**Total Landed Cost** - The complete cost to get goods to your door: CIF + duty + VAT.

## Calculating Your Total Landed Cost

**Step 1**: Determine CIF value
CIF = (Invoice Value × Exchange Rate) + Freight + Insurance

**Step 2**: Calculate import duty
Duty = CIF × Duty Rate
Use preferential rate if goods qualify under a trade agreement.

**Step 3**: Calculate VAT base
VAT Base = CIF + Duty

**Step 4**: Calculate import VAT
VAT = VAT Base × VAT Rate (20% standard)

**Step 5**: Total landed cost
Total = CIF + Duty + VAT

## Formula Reference

| Calculation | Formula |
|-------------|---------|
| CIF Value | Invoice x Exchange Rate + Freight + Insurance |
| Duty Amount | CIF x Duty Rate / 100 |
| VAT Base | CIF + Duty |
| VAT Amount | VAT Base x VAT Rate / 100 |
| Total Cost | CIF + Duty + VAT |
| Duty Per Unit | Duty Amount / Quantity (if applicable) |

## Common VAT Rates

- **Standard rate: 20%** — Most goods
- **Reduced rate: 5%** — Energy-saving materials, children's car seats
- **Zero rate: 0%** — Books, most food, children's clothes

## Data Sources

- **Duty rates**: UK Global Tariff (trade-tariff.service.gov.uk)
- **Exchange rates**: Bank of England daily rates
- **Trade agreements**: gov.uk/guidance/uk-trade-agreements
- **Rules of origin**: gov.uk/guidance/rules-of-origin-for-goods-moving-between-the-uk-and-eu

## Tips for Accuracy

1. **Always verify your commodity code** — The first 6 digits are global, but UK uses up to 10 digits for specific rates.

2. **Check preferential eligibility** — Goods must meet rules of origin (sufficient processing or value added in the partner country).

3. **Use the exchange rate** from the date of import, not the invoice date.

4. **Include all costs** in CIF: freight from port of export to UK, insurance during transit.

5. **VAT on imports** can be reclaimed if you're VAT registered and the goods are for business use.
