## What's Inside

### 1. UK Import Duty Calculator (Google Sheets)
The same duty calculator we sell standalone — calculates duty, VAT, and total landed cost for any UK import. Includes 200+ commodity code rates and trade agreement preferences.

### 2. Import Document Checklist (Google Sheets)
A reusable checklist template for every import shipment. Tracks:
- Commercial invoice received and checked
- Packing list verified against order
- Bill of lading / airway bill on file
- Certificate of origin (where needed for preferential rates)
- Import licence / IPAFFS notification (for controlled goods)
- Insurance certificate
- CDS declaration submitted
- Duty payment confirmed

Tick off each document per shipment. Never get caught at the border with missing paperwork.

### 3. Supplier Due Diligence Scorecard (Google Sheets)
Evaluate overseas suppliers before you commit. Score each supplier across:
- Product quality and consistency
- Lead time reliability
- Documentation accuracy (do they send correct invoices and COOs?)
- Compliance history (sanctions, restricted party screening)
- Communication and responsiveness
- Financial stability indicators

Compare suppliers side by side. Make data-driven sourcing decisions.

### 4. CDS Import Declaration Quick Reference (PDF)
A 12-page visual guide to completing a UK customs declaration on CDS. Covers:
- The 8 key data elements you must get right
- Header vs item level fields explained
- Procedure codes: which ones to use and when
- Common errors that trigger HMRC queries
- Step-by-step screenshots of the declaration process

Not a replacement for a customs broker — but enough to understand what your broker is doing and catch mistakes before they cost you money.
