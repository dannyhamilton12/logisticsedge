## Key Takeaways

- **Clothing duty rates vary from 6.5% to 12%** depending on fabric type and construction
- **Cotton garments face 12% duty**, while synthetic clothing attracts 6.5-8%
- **20% VAT applies** to all clothing imports over £135, calculated on duty-inclusive value
- **Trade agreements with Turkey and other partners** can eliminate or reduce duty rates
- **Rules of origin compliance** is critical for claiming preferential tariff treatment

---

Understanding UK import duty on clothing is essential for fashion retailers, e-commerce sellers, and anyone bringing textile products into the UK market. With duty rates varying significantly based on fabric composition and construction methods, getting the classification right can save thousands of pounds on larger shipments.

## UK Import Duty Rates for Clothing

### Standard Clothing Categories

**Cotton Garments (12% Duty)**
- Men's suits and jackets: 12% (HS Code 6203.11-6203.19)
- Women's dresses and skirts: 12% (HS Code 6204.41-6204.49)
- T-shirts and casual wear: 12% (HS Code 6109.10.00)
- Jeans and trousers: 12% (HS Code 6203.42.31)
- Children's cotton clothing: 12% (HS Code 6111.20.00)

**Synthetic Garments (6.5-8% Duty)**
- Polyester jackets: 6.5% (HS Code 6203.43.90)
- Synthetic dresses: 6.5% (HS Code 6204.44.00)
- Nylon activewear: 6.5% (HS Code 6211.43.00)
- Fleece and technical wear: 8% (HS Code 6210.40.00)

**Wool Garments (8-12% Duty)**
- Pure wool suits: 12% (HS Code 6203.12.00)
- Wool knitwear: 12% (HS Code 6110.11.00)
- Wool blend garments: 8% (HS Code 6203.13.00)

**Silk and Linen (6.5-8% Duty)**
- Silk blouses and dresses: 6.5% (HS Code 6206.30.00)
- Linen shirts: 8% (HS Code 6205.30.00)
- Mixed linen garments: 8% (HS Code 6211.42.00)

### Specialised Clothing Categories

**Underwear and Hosiery**
- Cotton underwear: 6.5% (HS Code 6108.21.00)
- Synthetic underwear: 6.5% (HS Code 6108.22.00)
- Tights and stockings: 8% (HS Code 6115.95.00)
- Sports bras: 6.5% (HS Code 6212.10.00)

**Footwear (Variable Rates)**
- Leather shoes: 8-17% depending on construction (HS Code 6403)
- Synthetic footwear: 4-16.9% (HS Code 6404)
- Sports shoes: 16.9% (HS Code 6404.11.00)
- Children's shoes: 7-12% (HS Code 6403.91)

**Accessories**
- Leather handbags: 3% (HS Code 4202.21.00)
- Textile bags: 2.7% (HS Code 4202.92.00)
- Hats and caps: 5.7% (HS Code 6505.00.00)
- Gloves: 6.5-8.5% depending on material (HS Code 6116)

For detailed classification guidance, refer to the [UK commodity codes guide](/articles/uk-commodity-codes-tariff-classification-guide/).

## Import VAT on Clothing

### VAT Calculation Method

**All clothing imports over £135 face 20% VAT**, calculated on the duty-inclusive value:

**Example Calculation:**
- Clothing value: £5,000
- Shipping cost: £200
- Insurance: £50
- **Customs value: £5,250**
- Import duty (12% on cotton): £630
- **Duty-inclusive value: £5,880**
- **VAT (20% of £5,880): £1,176**
- **Total charges: £1,806**

### VAT Relief Options

**Returned Goods Relief**
- Clothing exported from the UK and returned within 3 years
- Must prove goods were originally UK-manufactured or duty-paid
- Common for fashion shows and temporary exhibitions

**Temporary Admission Relief**
- Fashion samples for trade shows and buyer meetings
- 100% duty and VAT suspension for up to 24 months
- Requires ATA Carnet or customs procedure code

## Trade Agreement Benefits for Clothing

### EU-UK Trade and Cooperation Agreement

**Qualifying Origins:**
- EU-made clothing maintains preferential access under specific rules
- Complex rules of origin for textile products require careful documentation
- "Double transformation rule" often applies to manufactured garments

**Key Requirements:**
- Fabric must be woven/knitted in EU or UK
- Garment assembly must occur in EU or UK
- Qualifying materials from other preference-eligible countries permitted

### UK-Turkey Customs Union Arrangements

**Significant Duty Savings:**
- Turkish clothing often qualifies for **0% duty** under customs union arrangements
- Simpler origin requirements than EU rules
- Major advantage for cotton garments (normally 12% duty)

**Documentation Requirements:**
- EUR.1 movement certificate or registered exporter declaration
- Proof of Turkish manufacture and qualifying inputs
- Transport documents showing direct shipment

### Other Bilateral Agreements

**UK-Japan Comprehensive EPA**
- Limited clothing coverage but includes some technical textiles
- Useful for specialised sportswear and outdoor clothing
- Staged tariff elimination over multiple years

**UK-South Korea FTA**
- Maintains preferential access for Korean textile products
- Important for technical fabrics and synthetic garments
- Complex rules of origin requirements

For complete information on trade agreements, review the [import duty UK guide](/articles/import-duty-uk-complete-guide/).

## Fabric Classification and Duty Implications

### Material Composition Rules

**Determining Primary Fabric (>50% Rule):**
- Clothing duty rate determined by predominant fabric by weight
- Cotton >50% = 12% duty rate applies to entire garment
- Synthetic >50% = 6.5-8% duty rate

**Mixed Fabric Garments:**
- If no single material exceeds 50%, highest applicable rate applies
- Common with cotton-polyester blends
- Can significantly impact duty calculations

### Common Classification Mistakes

**Activewear vs Casual Wear**
- Sports-specific clothing may qualify for different rates
- Garment construction and intended use affects classification
- Marketing claims don't determine customs classification

**Children's vs Adult Sizing**
- Children's clothing (sizes up to age 14) often has different rates
- Size ranges must match official customs definitions
- Age-appropriate styling requirements apply

## Common Pitfalls and Compliance Issues

### 1. Undervaluation Risks

**HMRC Monitoring:**
- Clothing imports are closely monitored for undervaluation
- Comparison with published wholesale price databases
- Penalties up to 100% of evaded duty

**Valuation Method Requirements:**
- Transaction value must include all costs to UK border
- Royalty and licence fee inclusion rules
- Related party transaction adjustments

### 2. Origin Documentation Failures

**Incomplete EUR.1 Certificates:**
- Missing or incorrect supplier declarations
- Insufficient proof of qualifying processing
- Retroactive origin claims generally not permitted

**Rules of Origin Compliance:**
- Double transformation requirements for many garments
- Qualifying percentage thresholds for mixed materials
- Non-originating material limitations

### 3. Product Safety Non-Compliance

**Children's Clothing Violations:**
- Drawstring length restrictions regularly violated
- Small parts detachment during testing
- Fire safety standard failures for sleepwear

**Chemical Restrictions:**
- Azo dye prohibitions in direct skin contact clothing
- Heavy metal content limits in accessories
- REACH registration requirements for treatment chemicals

## Practical Examples: Real Import Scenarios

### Example 1: Online Fashion Retailer

**Scenario:** UK e-commerce company imports mixed clothing from China
- Cotton t-shirts value: £8,000
- Synthetic dresses value: £12,000
- Shipping and insurance: £600
- **Total customs value: £20,600**

**Customs Calculation:**
- Cotton t-shirts duty (12%): £960
- Synthetic dresses duty (6.5%): £780
- Total duty: £1,740
- VAT base (£20,600 + £1,740): £22,340
- **VAT (20%): £4,468**
- **Total charges: £6,208**

### Example 2: Turkish Clothing Import

**Scenario:** Fashion wholesaler imports from Turkey using preferential origin
- Cotton garments value: £15,000
- Shipping and insurance: £400
- **Total customs value: £15,400**

**With Turkish Preference:**
- Duty rate: 0% (customs union benefit)
- Customs duty: £0
- **VAT (20% of £15,400): £3,080**
- **Total charges: £3,080**

**Without Preference:**
- Duty (12%): £1,848
- VAT base: £17,248
- VAT (20%): £3,450
- **Total charges: £5,298**
- **Savings with preference: £2,218**

## Frequently Asked Questions

**Q: Do vintage or second-hand clothes have different duty rates?**
A: No, used clothing uses the same commodity codes and duty rates as new items. The condition doesn't affect classification.

**Q: How is duty calculated on mixed fabric garments?**
A: Duty is based on the predominant fabric by weight. If no single material exceeds 50%, the highest applicable rate applies to the entire garment.

**Q: Can I claim duty relief for clothing samples?**
A: Yes, genuine trade samples may qualify for duty relief under specific conditions, including value limits and marking requirements.

**Q: What happens if my Turkish supplier doesn't have EUR.1 certificates?**
A: Without proper origin documentation, you'll pay the full UK Global Tariff rate. Retroactive preference claims are generally not allowed.

**Q: Are designer handbags subject to higher duty rates?**
A: No, duty rates are based on construction and materials, not brand value. However, luxury goods may face additional scrutiny for valuation accuracy.

**Q: How do clothing returns affect duty payments?**
A: Returned goods may qualify for duty relief if they meet strict conditions, including time limits and proof of original UK duty payment.
