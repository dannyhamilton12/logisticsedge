# LogisticsEdge Weekly Newsletter Template Guide

## Overview

Professional email newsletter template designed for LogisticsEdge Weekly, targeting UK logistics and customs professionals. Built for Buttondown platform with mobile-responsive design and brand-aligned styling.

## Template Features

### ✅ Brand Alignment
- Uses LogisticsEdge color palette (Navy #0f172a, Accent #3b82f6)
- Inter font family throughout
- Category-specific color coding
- Professional, no-fluff design aesthetic

### ✅ Content Structure
- **Lead Story Section**: Featured article or major industry development
- **Regulatory Roundup**: Compliance changes, new regulations, policy updates
- **Industry Insights**: Market trends, analysis, expert perspectives
- **Essential Reads**: Curated guides, tools, and resources
- **Quick Links**: Fast access to popular LogisticsEdge content

### ✅ Technical Specifications
- Mobile-responsive design (breakpoint: 480px)
- Email client compatibility (Outlook, Gmail, Apple Mail, etc.)
- Maximum width: 600px for optimal readability
- Bulletproof buttons and progressive enhancement
- MSO conditional comments for Outlook support

### ✅ Category System
- **CUSTOMS** (Purple #8b5cf6): Import duty, tariffs, clearance
- **LOGISTICS** (Blue #0ea5e9): 3PL, transportation, operations
- **TRADE** (Orange #f97316): Incoterms, international commerce
- **SUPPLY CHAIN** (Green #10b981): End-to-end logistics processes
- **WAREHOUSING** (Indigo #6366f1): Storage, fulfilment, distribution

## Using with Buttondown

### 1. Template Setup
1. Copy the HTML content from `newsletter-template.html`
2. In Buttondown dashboard, go to Settings > Email Design
3. Select "Custom HTML" template option
4. Paste the template code
5. Save as "LogisticsEdge Weekly Template"

### 2. Content Variables
Replace these placeholders when creating each newsletter:

#### Header Variables
- `[ISSUE_NUMBER]` - Sequential issue number (e.g., "47")
- `[DATE]` - Publication date (e.g., "4 April 2026")

#### Lead Story Variables
- `[LEAD_STORY_TITLE]` - Main headline
- `[LEAD_STORY_SUMMARY]` - 2-3 sentence summary
- `[LEAD_STORY_LINK]` - URL to full article

#### Regulatory Section Variables
- `[REGULATION_TITLE_1/2]` - Regulation headline
- `[REGULATION_SUMMARY_1/2]` - Brief explanation
- `[EFFECTIVE_DATE]` / `[DEADLINE_DATE]` - Key dates
- `[REGULATION_LINK_1/2]` - Links to detailed coverage

#### Industry Insights Variables
- `[INSIGHT_TITLE_1/2]` - Analysis headline
- `[INSIGHT_SUMMARY_1/2]` - Key takeaways
- `[INSIGHT_LINK_1/2]` - Links to full analysis

#### Essential Reads Variables
- `[RESOURCE_TITLE_1/2]` - Resource name
- `[RESOURCE_SUMMARY_1/2]` - Value proposition
- `[RESOURCE_LINK_1/2]` - Access links

#### Footer Variables
- `[UNSUBSCRIBE_URL]` - Buttondown unsubscribe link
- `[MANAGE_URL]` - Preference management link

## Content Guidelines

### Tone & Style
- **Professional**: Business language appropriate for senior logistics professionals
- **Direct**: Get to the point quickly, no unnecessary filler
- **UK-focused**: Emphasis on UK regulations, markets, and implications
- **Actionable**: Include practical steps or clear implications

### Article Summaries
- **Length**: 25-40 words per summary
- **Format**: State the key point, implication, and timeline if relevant
- **Value**: Focus on "what this means for you" rather than just "what happened"

### Headlines
- **Clarity**: Immediately clear what the story covers
- **Relevance**: Obvious connection to UK logistics professionals
- **Urgency**: Indicate time-sensitivity where applicable
- **Specificity**: Avoid vague terms, use precise language

### Links Strategy
- Always link to LogisticsEdge content where available
- For external links, prefer government sources, industry bodies, major publications
- Include context in link text (avoid "click here" or "read more" without context)
- Ensure all links are tested and working
